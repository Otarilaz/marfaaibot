# marfa-md-backup

Бэкап markdown-документации по проекту **Marfa** (срез от `2026-03-31`).

## Что внутри

- `marfa-ai/`
  - `docs/` — документация (в т.ч. `max_bot.latest.md`)
  - `API_KEYS_SETUP.md` — инструкция куда класть ключи (**ключи в репозиторий не добавлять**)
- `marfa-landing/`
  - `docs/` — документация по лендингу/операциям

## Как публиковать на GitHub

1. Убедиться, что в репозитории нет `.env` и `node_modules/` (см. `.gitignore`).
2. Залить в GitHub как обычный репозиторий.
